1. Login to the [SanoServers Control Panel](https://panel.sanoservers.com/)
2. Navigate to your server's control panel


There are two ways to update your Minecraft server:

### Method 1: Updating via the Control Panel
1. Select the Versions tab
2. Select the version you want to update to
3. Select the Minor version you want to update to
4. Click `Install`

### Method 2: Updating Manually
1. Stop your server
2. Download the latest server version from the [Minecraft website](https://www.minecraft.net/en-us/download/server)
3. Upload the server file to your server's root directory
4. Restart your server to apply the update

Your Minecraft server should now be updated to the latest version. If you encounter any issues, feel free to reach out to our support team for assistance.
